プライバシーポリシー / 개인정보처리방침

アプリ名 / 앱 이름: hikaro
運営者 / 운영자: 조장연 (Jangyean Cho)
最終更新日 / 최종 업데이트: 2026年3月

1. 収集する情報 / 수집하는 정보

本アプリは、以下の情報を収集することがあります。

  • ログイン情報（Apple・Google・LINEアカウントを通じて取得される識別子・メールアドレス）
  • 購読状態（有料・無料）
  • 学習進捗データ（端末内、およびログイン時にサーバーへ同期）
  • アプリのクラッシュログ・利用状況（匿名）

本アプリは、氏名・電話番号などの情報を独自に収集しません。

2. 情報の利用目的 / 정보의 이용 목적

収集した情報は、以下の目的にのみ使用します。

  • アプリ機能の提供・改善
  • 学習データの復元（再インストール・機種変更時）
  • 購読サービスの管理
  • 不具合の修正・サポート対応

3. 第三者への提供 / 제3자 제공

収集した情報を、以下の場合を除き第三者に提供することはありません。

  • 法令に基づく開示が必要な場合
  • ユーザーの同意がある場合

なお、本アプリは以下のサービスを利用しており、それぞれのプライバシーポリシーが適用されます。

サービス目的プライバシーポリシー
Apple サインインログイン認証apple.com/legal/privacy
Google ログインログイン認証policies.google.com/privacy
LINE ログインログイン認証line.me/ja/terms/policy
RevenueCat購読管理revenuecat.com/privacy

4. 決済情報について / 결제 정보

決済はすべてApp Store（Apple）またはGoogle Play（Google）を通じて処理されます。
運営者はカード番号などの決済情報を直接収集・保管しません。

5. データの保管 / 데이터 보관

  • 学習進捗データは端末内に保存されます。
  • ログイン時は、日本国内のサーバーにも同期されます（アプリの再インストールや機種変更後もデータを復元するため）。
  • 氏名・連絡先などの個人情報はサーバーに保管しません。

6. お子様のプライバシー / 아동 개인정보

本アプリは13歳未満の方を対象としておりません。
13歳未満の方の個人情報を意図的に収集することはありません。

7. プライバシーポリシーの変更 / 방침 변경

本ポリシーは予告なく変更される場合があります。
重要な変更がある場合は、アプリ内またはストアページにてお知らせします。
